# Python 3 Learning Project

## Project
Python 3 learning repo following a Polish Udemy course:
"kurs-python-od-podstaw-dla-poczatkujacych-od-zera-do-mastera"

- Python 3.13, managed with `uv` (no external dependencies)
- Run any file: `uv run python <path/to/file.py>`

## Structure
```
basics/01_data_types/   — 8 lesson files (01–07, plus 05-1)
basics/02_operators/    — empty, next lesson area
tasks/                  — 8 practical exercises (01–08)
challenges/             — 9 challenge exercises (01–07 + extras)
tips/                   — markdown reference notes
```

## Conventions
- Files named `NN_topic_name.py` (sequential numbering)
- Comments and task descriptions in Polish (course language)
- Modern Python style: f-strings, type hints
- Each file is self-contained and executable with inline output examples
- Challenges numbered to match corresponding lesson
